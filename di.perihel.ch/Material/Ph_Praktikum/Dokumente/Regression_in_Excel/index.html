Regression mit Excel

An einem Beispiel sollen Sie lernen, wie man mit Excel eine Kurvenanpassung (Regression, Fit) durchführt. Die Daten sind Massen und Werte von Eurocent-Münzen. Sie sollen untersuchen, ob und wie die Masse vom Wert abhängt. Das folgende Totorial wurde auf einem Macintosh-Computer erstellt, aber die Befehle lauten in Windows ähnlich. Probieren Sie die Anleitung Schritt für Schritt selber aus. Als erstes müssen Sie Excel starten.
 

Abbildung 1: Geben Sie die Daten in Excel ein, die Münzwerte als erste Spalte (entspricht x) und die Münzmassen als 2. Spalte (entspricht y).
 
Abbildung 2: Markieren Sie die Spalten mit den Daten und wählen Sie "Diagramm einfügen".
 
Abbildung 3: Stellen Sie das Diagramm korrekt dar, dann wählen Sie unter dem Menüpunkt Diagramm "Trendlinie einfügen" (das Diagramm vorher anklicken).
 
Abbildung 4: Wählen Sie einen passenden Trendlinien-Typ, hier z.B. eine Potenzfunktion.
 
Abbildung 5: Lassen Sie unter "Optionen" die Gleichung der Regressionsfunktion (Trendlinie) sowie ev. das Bestimmtheitsmass im Diagramm anzeigen.
 
Abbildung 6: Jetzt sollte dieses Bild erscheinen.
 
Abbildung 7: Falls die Parameter der Regressionsfunktion mit zuwenig Stellen angezeigt werden, so klicken Sie auf die Trendliniengleichung im Diagramm und wählen unter "Format" den Punkt "markierte Datenbeschriftung".
 
Abbildung 8: Wählen Sie sodann im Zahlenformat die passende Anzahl Stellen aus.
 
Und falls es nicht geklappt hat, können Sie das Excelfile herunterladen und damit herumprobieren: Cent_Masse.xls
