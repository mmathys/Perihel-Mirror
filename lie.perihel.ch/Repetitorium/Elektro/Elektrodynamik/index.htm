Elektrodynamik

magnetische Induktion
Folgende Effekte zeigen sich in einfachen Demonstrationsversuchen. Die Effekte werden elektromagnetische Induktion (Michael Faraday, 1820) genannt.

Schlägt man ein Kupferkabel zwischen die Schenkel eines Hufeisenmagneten oder bewegt man einen Stabmagneten in eine Schlaufe aus unmagnetischem, leitendem Material, so kann man an den Enden des Leiters kurzzeitig eine Spannung messen. Je schneller die Bewegung ist, desto grösser wird die Spannung.

Die Lorentzkraft treibt die freien Ladungsträger im Leiter an. Bewegt sich ein Leiter der Länge L mit Geschwindigkeit v senkrecht zu den Feldlinien durch ein Magnetfeld mit Flussdichte B, so wird eine Spannung der Grösse Uind = vBL induziert.



Vergrössert man die Fläche einer Drahtschlaufen, indem man wie im Bild einen Leiter durch ein Magnetfeld bewegt, so kann man während der Bewegung eine Spannung messen. Auch hier ist die Lorentzkraft die tiefere Ursache.


Hält man eine Spule über eine Magnetspule, die von Wechselstrom durchflossen wird und somit ein magnetisches Wechselfeld erzeugt, so kann man zwischen den Anschlüssen der oberen Spule eine Wechselspannung messen. Die Spannung ist um so grösser, je stärker das Magnetfeld ist und je schneller es sich ändert. Dieser Versuch zeigt das Grundprinzip des Transformators.

Die induzierte Spannung ist proportional zur Zahl der Windungen N, zur Fläche einer Windung A und zur Änderungsrate der Flussdichte dB/dt:
Uind = NA·dB/dt


Rotiert eine Flachspule gleichmässig in einem homogenen Magnetfeld, so kann zwischen den Anschlüssen der Rotorspule eine Wechselspannung gemessen werden. Dieser Versuch zeigt das Grundprinzip des Generators.

Die induzierte Spannung (Spitzenwert) ist proportional zur Windungszahl N der Spule, der Fläche A einer Windung, zur Winkelgeschwindigkeit ω und der Flussdichte B:
Uind,max = NAωB
(Die Drehachse muss senkrecht zu den Feldlinien stehen.)


Die oben dargestellten Phänomene werden im Faraday'schen Induktionsgesetz zusammengefasst. Dieses Gesetz verwendet den sog. magnetischen Fluss Φ

Definition: magnetischer Fluss Φ = BsA
Einheit: T m2 = V s = Wb (Weber)
Der magnetische Fluss durch eine Fläche A ist das Produkt aus Flächeninhalt und Komponente Bs der Flussdichte B senkrecht zur Fläche. Diese Definition gilt für ein homogenes Feld und eine ebene Fläche. Bei inhomogenen Feldern oder gekrümmten Flächen muss man die Fläche in kleine Stücklein aufteilen und dann die Teilflüsse summieren.

Der magnetische Fluss durch eine geschlossene Fläche verschwindet. Umschliesst man einen Magneten mit z.B. einer Kugelfläche, so tritt jede Feldlinie, welche die Kugel verlässt, an anderer Stelle wieder ein. Die entsprechenden Flussanteile kompensieren sich.

Faraday'sches Induktionsgesetz
Uind = -dΦ/dt oder Uind = -ΔΦ/Δt
Die Induktionsspannung ist gleich der Flussänderung pro Zeit; das Vorzeichen erinnert an die Lenz'sche Regel (s. unten) und ist meist ohne Belang.

Induktionsströme
An einer offenen Leiterschleife kann man eine Induktionsspannung messen, in einer geschlossenen Schleife einen Strom.
Lenz'sche Regel: Der Induktionsstrom ist so gerichtet, dass er seiner Ursache entgegen wirkt. (Deshalb das Minuszeichen im Induktionsgesetz).
Bewegt man eine geschlossene Leiterschleife in ein Magnetfeld hinein, so entstehen Induktionsströme. Auf diese Ströme wirken Biot-Savart-Kräfte, welche diese Bewegung bremsen. (Eine antreibende Wirkung würde den Energiesatz verletzen.)
Wird ein Kupferblech in ein Magnetfeld hinein bewegt, so entstehen Wirbelströme. Die magnetischen Kräfte auf die Wirbelströme bremsen die Bewegung während des Eintritts und Austritts.

Ein änderndes Magnetfeld erzeugt elektrische Wirbelfelder.
